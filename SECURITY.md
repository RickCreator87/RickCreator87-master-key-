## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

The maintainers of **RickCreator87-master-key** take security issues seriously. If you believe you have found a security vulnerability, please report it to us as described below.

### **Private Disclosure Process**
1.  **Email**: Please report suspected security vulnerabilities by sending an email to `<ricks.online.businesses425@gmail.com@www.gitdigital.ai>`.
2.  **Information Required**: Include the following details in your report:
    *   The project version (e.g., commit hash, branch name).
    *   A detailed description of the vulnerability.
    *   Steps to reproduce the issue.
3.  **Response**: You will receive an acknowledgment within **48 hours**. The security team will provide a more detailed response and work with you to understand and address the issue.
4.  **Updates**: After the initial report, the team will keep you informed about the progress toward a fix and a public announcement.

We appreciate your efforts to disclose your findings responsibly and will make every effort to acknowledge your contributions if the vulnerability is confirmed and fixed.
